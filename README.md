🏠 My Airbnb Backend + Frontend

This is a full-stack Airbnb clone project built using modern web technologies.

🚀 Features
User authentication (Login/Signup)
Property listings
Backend API with Express
Frontend built with React
Responsive UI
🛠 Tech Stack
Frontend: React (Vite)
Backend: Node.js, Express
Database: MongoDB
Deployment: Vercel
📂 Project Structure
client/ → Frontend (React)
server/ → Backend (Node.js & Express)
🌐 Live Demo

👉 https://my-airbnb-backend-isnm.vercel.app

⚙️ Installation
1. Clone the repo

git clone https://github.com/CSaditya7/my-airbnb-backend.git

2. Install dependencies

cd client
npm install

cd ../server
npm install

3. Run project

Frontend:
npm run dev

Backend:
npm start
